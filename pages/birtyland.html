Introdução

Originalmente desenvolvido para a Game Jam Ludum Dare 40. Shooter top-down com alguns conceitos de RPG e MOBA. O objetivo é ganhar vários pontos destruído os inimigos e suas colméias para evoluir seus atributos e lutar contra o chefe que surge a cada minuto. A versão HTML5 possui controles via toque quando acessado via navegador mobile (recomendado uso do Firefox).

Informações

Há lista de recordes para os melhores tempo. A versão Windows possui opção para modificar o tamanho da janela. O redimensionamento é feito multiplicando a resolução base e mínima que é 160x144. A versão Android possui um sistema de anúncios (Unity ADS) que exibe um vídeo quando o jogador inicia uma nova partida, caso haja um tempo mínimo desde o último anúncio.

Projeto criado para a categoria Compo da Ludum Dare 40. Nela, os participantes têm 48 horas para criar um jogo com o tema em questão, sendo obrigatório criar todos os gráficos e áudio durante o evento. O tema escolhido foi: "The more you have, the worse it is". Com base no feedback recebido, desenvolvi uma versão revisada.

Minha participação

Desenvolvi tudo. Gráficos criados com o Paint.NET. Áudio criado com o BeepBox. Utiliza o sistema de tilemap da Unity
