Introdução

Quiz interativo para treinamento de sinalização segura da Arteris para acidentes.

Informações

O jogador precisa seguir as instruções das cartas. Elas explicam onde colocar cones e veículos para uma sinalização segura.

Minha participação

Fui responsável por toda a parte de programação do jogo. As ações de cada carta são definidas utilizando o sistema de eventos da Cocos. O projeto foi integrado com SCORM
