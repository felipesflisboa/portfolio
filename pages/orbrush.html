Introdução

Originalmente desenvolvido para a GMTK 2020 Game Jam. Jogo de corrida de orbes com cartas de efeito. Inspirado em corridas de bolas de gudes.

Informações

Projeto criado em 48 horas para a Game Jam cujo tema escolhido foi "out of control". Com base no feedback recebido, desenvolvi uma versão revisada. Alguns sistemas:

  • Modo Marathon: O jogo vai ficando mais difícil para cada nível vencido. Possui lista de recordes dos melhores níveis alcançados.
  • Modo Quick Race: Partidas com diferentes dificuldades configuráveis. A versão WebGL possibilita multiplayer com bots. O jogador pode escolher usar mouse, teclado ou joystick.
  • Sistema de anúncios (Unity Ads) que exibe um vídeo quando o jogador inicia uma nova partida, caso haja um tempo mínimo desde o último anúncio (versão Android).

Minha participação

Desenvolvi tudo exceto a parte sonora, o core do Game Design, o gráfico da pista, das orbes, das cartas e partículas.
