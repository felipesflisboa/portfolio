Introdução

Originalmente desenvolvido para a Nokia Game Jam 2. Game de ação em que o jogador precisa abduzir animais enquanto desvia dos tiros.

Informações

O jogo conta com teclado virtual para inputs e o maior recorde é salvo.

Desenvolvido utilizando a versão 0.14.3 do Project Tiny da Unity que usa o ECS (Entity Component System), cujo objetivo é criar jogos otimizados/leves, possibilitando a versão web funcionar em dispositivos mobile. Foi portado de HTML5 para Android por meio do Apache Cordova.

Projeto criado para a Nokia Game Jam, cuja proposta é fazer um jogo com resolução 84x48, só usando duas cores e som monofônico, remetendo aos jogos do Nokia 3310. Participou da segunda edição deste evento, no qual ficou rankeado na 28 posição em Design e 32 em satisfação dentre os 140 participantes. Com base no feedback recebido, desenvolvi uma versão revisada.
