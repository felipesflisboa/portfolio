Introdução

Jogo de combate entre tanques com vista de cima. A mira segue o mouse e a movimentação é realizada via teclado. O destaque são as balas que quicam na parede, tornando necessários desviar dos seus próprios projéteis. Possui Single Player e Multiplayer para até quatro jogadores via rede.

Informações

Os dois primeiros modos de jogo contam cada um como uma lista de recordes. Os modos são:

  • Arcade: O jogador deve vencer quatro fase, cada um com quatro subfases e um chefe.
  • Survival: Combates consecutivos até o jogador ser derrotado. Não é possível recuperar a energia.
  • Single Duel: O jogador pode selecionar para batalhas qualquer arena e de um a três oponentes.
  • Multiplayer Duel: Combates mutiplayer. Feito utilizando o primeiro sistema de Network da Unity.
